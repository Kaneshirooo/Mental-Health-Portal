Conceptual Framework of the Study

This framework presents the Mental Health Web Portal for Pangasinan State University San Carlos Campus through two complementary models: a System Map illustrating the actors and data flows, and an Input–Process–Output (IPO) diagram showing how user inputs become actionable wellness outputs.
